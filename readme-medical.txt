# Medical Supply Chain Optimization Dashboard

## Project Overview
This project creates an interactive dashboard for analyzing and optimizing medical supply chain operations. It focuses on key metrics including inventory levels, demand forecasting, supplier performance, and logistics efficiency.

## Features
- Real-time inventory tracking and demand analysis
- Supplier performance metrics
- Stockout risk assessment
- Automated demand forecasting
- Interactive visualizations using Plotly
- Comprehensive insights generation

## Visualizations
1. Inventory vs Demand Trends
2. Supplier Performance Analysis
3. Product Stockout Risk Assessment
4. Delivery Time Analysis

## Requirements
- Python 3.7+
- Required packages listed in requirements.txt

## Installation & Usage

1. Clone this repository:
```bash
git clone https://github.com/yourusername/medical-supply-dashboard.git
cd medical-supply-dashboard
```

2. Install required packages:
```bash
pip install -r requirements.txt
```

3. Run the dashboard:
```bash
python main.py
```

4. Open the generated `medical_supply_dashboard.html` file in your web browser to view the interactive dashboard.

## Output Files
- `medical_supply_dashboard.html`: Interactive dashboard with visualizations
- `supply_chain_insights.txt`: Detailed analysis and forecasts

## Data
The project uses synthetic data that simulates real medical supply chain patterns, including:
- Daily demand and inventory levels
- Supplier delivery times
- Product stockout risks
- Seasonal patterns and trends
